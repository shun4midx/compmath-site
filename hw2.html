Question 3b

First, we consider evaluating the condition number,

For $f(x) = \frac{e^x - 1}{x}$, we notice $f'(x) = \frac{xe^x - (e^x - 1)}{x^2}$.

By $\kappa_f(x) = |\frac{xf'(x)}{f(x)}|$, we get $\kappa_f(x) = |(\frac{xe^x - (e^x - 1)}{x}) \cdot (\frac{x}{e^x - 1})| = |\frac{xe^x}{e^x - 1} - 1|$.

Now, we consider what values of $x$ cause $\kappa_f(x) \to \infty$.

We notice, $\kappa_f(x) \to \infty$ when $|\frac{xe^x}{e^x - 1}| \to \infty$. When the denominator equals $0$ (and hence limit $\to \infty$), we have $e^x - 1 = 0 \Rightarrow x = 0$. Otherwise, as $x \to \infty$, then $|\frac{xe^x}{e^x - 1}| = |x + \frac{x}{e^x - 1}| \to \infty$. As $x \to -\infty$, by L'Hopital's Rule, $\lim_{x \to -\infty}|\frac{xe^x}{e^x - 1}| = \lim_{x \to -\infty}|\frac{e^x(x + 1)}{e^x}| = \infty$.

Hence, in conclusion, the condition number of $f(x)$ approaches $\infty$ when $\boxed{x = 0, x \to \pm \infty}$.

Question 6

WLOG, we pick one root called $r$, where $f(b) = r$.

By using implicit differentiation w.r.t. $b$, we get $ar^2 + br + c = 0 \Rightarrow 2ar (\frac{dr}{db}) + b\frac{dr}{db} + r = 0 \Rightarrow \frac{dr}{db} = -\frac{r}{2ar + b}$

Hence, the condition number for the problem $f(b) = r$ is $\kappa(b) = |\frac{b}{r} \cdot \frac{dr}{db}| = |\frac{b}{2ar + b}|$.

By substiuting $r = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$, we get $\kappa(b) = |\frac{b}{\sqrt{b^2 - 4ac}}|$.

Now, notice the roots of the original polynomial ($r_+$ and $r_-$) satisfy:

  • $r_+ + r_- = -\frac{b}{a}$
  • $r_+ - r_-$ = $\frac{\sqrt{b^2 - 4ac}}{a}$

Thus, $\kappa(b) = \boxed{\left|\frac{r_+ + r_-}{r_+ - r_-}\right|}$

# Polyadd
def polyadd(p, q):
    # Make sure deg p > deg q
    if len(p) < len(q):
        copy = p
        p = q
        q = copy
    
    # Iterate
    ans = []
    
    for i in range(len(p), 0, -1):
        ans = ans + [p[len(p) - i] + (0 if i > len(q) else q[len(q) - i])]

    return ans

# Samplevar
def samplevar(x):
    n = len(x)
    if (n <= 1):
        print("Invalid length of x.")
        return

Question 5

Part (a)

We see $f(x) = \frac{e^x - 1}{x}$, so $f'(x) = \frac{xe^x - (e^x - 1)}{x^2}$. Notice, $xf'(x) = e^x - f(x)$.

Hence, $\kappa_f(x) = \left|\frac{xf'(x)}{f(x)}\right| = \left|1 - \frac{xe^x}{e^x - 1}\right| = \left|(1 - x) - \frac{x}{e^x - 1}\right|$

Notice, over $-1 \leq x \leq 1$, we have $e^x - 1 \geq x$, which implies monoticity of $e^x - 1$, and moreover, $\frac{x}{e^x - 1} \leq 1$. Hence, the maximum value of $\kappa_f(x)$ must occur at the endpoints of the range $[-1, 1]$.

Notice,

  • $\kappa_f(-1) = \left|(1 - (-1)) - \frac{-1}{e^{-1} - 1}\right| = \left|2 + \frac{e}{1 - e}\right| = \frac{e - 2}{e - 1}$
  • $\kappa_f(1) = \left|(1 - 1) - \frac{1}{e^1 - 1}\right| = \frac{1}{e - 1}$

Notice, $\kappa_f(-1) < \kappa_f(1)$.

Hence, the maximum of $\kappa_f(x)$ is at $x = 1$, yielding $\boxed{\frac{1}{e - 1}}$.

import math
import numpy as np
import matplotlib.pyplot as plt

def naive(x):
    if x == 0:
        return 1.0
    return (math.exp(x) - 1) / x

def taylor(x):
    factorial = 1
    ans = 1

    for i in range(2, 18, 1):
        factorial *= i
        ans += 1 / factorial * (x ** (i - 1))

    return ans

# X grid (avoid exact division-by-zero issues by handling x==0 separately)
